# Visitrack Design System Mandates

All future UI development for this project must strictly adhere to the **Modern Monochrome Minimalist** (Swiss-Inspired) aesthetic established in February 2026.

## 1. Typography & Hierarchy
- **Main Font:** Inter (or clean system sans-serif).
- **Primary Headers:** `text-3xl font-bold text-gray-900 tracking-tight`.
- **Subheaders/Metadata:** `text-sm font-medium text-gray-500 uppercase tracking-wider`.
- **Registry Labels:** `text-[10px] font-bold text-gray-400 uppercase tracking-widest`.
- **Technical IDs:** Must use `font-mono` for alignment and precision.

## 2. Color Palette (Strictly Monochrome)
- **Primary Text:** `text-gray-900` (Almost black).
- **Secondary Text:** `text-gray-500` or `text-gray-400`.
- **Backgrounds:** `bg-white` for cards, `bg-gray-50/30` for subtle sectioning.
- **Accents:** Only Black (`bg-black`, `border-black`) and White. **No colors (RGB) are permitted** unless they are student profile photos.

## 3. Geometry & Spacing
- **Border Radius:** Standardize on `rounded-2xl` for containers and modals. Use `rounded-xl` for smaller elements like buttons or icons.
- **Borders:** Minimalist hairline borders using `border border-gray-100` or `border-2 border-black/5`.
- **Table Spacing:** Rows must have generous vertical padding (`py-5`) and horizontal margins (`px-8`).
- **Negative Space:** Prioritize whitespace to ensure data readability and a "luxury" feel.

## 4. Component Patterns
- **Cards:** White background, subtle gray border, high-contrast black icons in `rounded-xl` containers.
- **Buttons:** Solid black with white text or subtle gray ghost buttons. High-contrast only.
- **Search Bars:** `rounded-xl` with absolute-positioned icons and subtle `placeholder:text-gray-400`.

*These mandates take absolute precedence over general defaults.*
